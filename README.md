# peco_find

Search directores or files and show with [fd] and [peco].

| Key bind  | Description                |
|:---------:|:---------------------------|
| `C-x C-d` | Search directories         |
| `C-x C-f` | Search filies              |

[fd]: https://github.com/sharkdp/fd
[peco]: https://github.com/peco/peco
